# AI-Native Book with Embedded RAG Chatbot

Welcome to the AI-Native Book with Embedded RAG Chatbot project. This book covers foundational concepts for AI students and software engineers transitioning into Physical AI and Humanoid Robotics.

## Module 1: The Robotic Nervous System (ROS 2)

This module provides an introduction to ROS 2 as a robotic nervous system, covering:

1. **Chapter 1**: Introduction to ROS 2 fundamentals, architecture, and biological nervous system analogy
2. **Chapter 2**: ROS 2 communication primitives (nodes, topics, services, actions)
3. **Chapter 3**: Humanoid robot description with URDF

### Features

- Comprehensive coverage of ROS 2 concepts for humanoid robotics
- Clear explanations suitable for both developers and CS students
- Practical examples and conceptual diagrams
- Integration-ready for future RAG chatbot functionality

### Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Build for production:
   ```bash
   npm run build
   ```

### GitHub Pages Deployment

To deploy to GitHub Pages:
```bash
GIT_USER=<Your GitHub username> USE_SSH=true npm run deploy
```

### Forward Compatibility

The content structure is designed to support future integration with a RAG chatbot system as specified in the project constitution.
